# Hasura Docs Sample App
Sample Hasura model and schema structure for [documentation](https://hasura.io/docs/) of a fictitious e-commerce store. 

## One Click Deploy
Use this URL to deploy this app to [Hasura Cloud](https://cloud.hasura.io/)

[https://cloud.hasura.io/deploy?github_repo=https://github.com/hasura/docs-sample-app/&hasura_dir=/](https://cloud.hasura.io/deploy?github_repo=https://github.com/hasura/docs-sample-app/&hasura_dir=/)
